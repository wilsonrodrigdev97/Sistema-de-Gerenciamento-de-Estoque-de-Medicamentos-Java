Conceitos Aplicados:
Manipulação de Arrays: Uso de vetores paralelos para gerenciar nomes de produtos e quantidades em estoque.

Modularização com Métodos: Divisão de responsabilidades entre métodos de exibição (mostrarEstoque) e processamento de lógica (atualizarQuantidade).

Estruturas de Repetição: Implementação de um loop infinito (while true) para manter o sistema operacional até a intervenção do usuário.

Tratamento de Fluxo: Validações de entrada (IDs válidos) e verificações de regras de negócio (estoque insuficiente).
